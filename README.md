# 🛒 Amazon Clone – JavaScript E-commerce Website

This project is a simplified Amazon-like e-commerce website built using HTML, CSS, and JavaScript.  
It was developed by following the SuperSimpleDev JavaScript tutorial to strengthen core frontend development skills.

---

## 🚀 Features

- 🛍️ Product listing page (Amazon-style layout)
- ➕ Add to Cart functionality
- 🛒 Cart page with item management
- 🔢 Dynamic cart quantity updates
- 💰 Price calculation and order summary
- 🔄 Interactive UI using JavaScript DOM manipulation

---

## 🧩 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📚 Learning Objectives

This project was built to practice and understand:

- JavaScript fundamentals (variables, functions, arrays, objects)
- DOM manipulation and event handling
- Building interactive web applications without frameworks
- Managing application state (cart system)
- Structuring a real-world UI layout

---

## ⚙️ Project Structure

- `index.html` – Main product listing page
- `cart.html` – Shopping cart page
- `scripts/` – JavaScript logic (cart, products, etc.)
- `styles/` – CSS styling files
- `images/` – Product images

---

## 🏃‍♂️ How to Run

1. Clone the repository or download ZIP  
2. Open `index.html` in your browser  
3. Start adding items to the cart
